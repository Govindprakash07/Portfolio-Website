# Personal Portfolio Website 🌐  

A simple personal portfolio website built using Node.js, Express, and EJS.  

## Features  
✔️ Showcases my projects  
✔️ Contact form  
✔️ Simple and clean user interface  

## Technologies Used  
- Node.js  
- Express.js  
- EJS for templating
